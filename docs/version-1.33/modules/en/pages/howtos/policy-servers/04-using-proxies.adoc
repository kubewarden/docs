= Configuring PolicyServers with proxies
:revdate: 2026-02-24

PolicyServers respect the standard environment variables for dealing with proxy
setups. The policy-server uses the configured proxy when downloading policies.
This includes Sigstore operations when verifying policies and images.

These variables are:

* `HTTP_PROXY` or `http_proxy`: proxy server for HTTP requests
* `HTTPS_PROXY` or `https_proxy`: proxy server for HTTPS requests
* `NO_PROXY` or `no_proxy`: comma-separated list of hosts to exclude from
  proxying

Please refer to the
xref:/reference/CRDs.adoc#k8s-api-github-com-kubewarden-kubewarden-controller-api-policies-v1-policyserverspec[`PolicyServer.spec.env`] field of
the CRD to set these environment variables.
